# Energy (NSW) Research Code

This repository contains research code for analyzing, preprocessing, and forecasting Australian National Electricity Market (NEM) prices, with a specific focus on the New South Wales (NSW) region. It features both classical data analysis approaches and advanced machine learning modeling, including a custom PyTorch-based diffusion model.

## Project Structure

### Data & Results
- **`data/`**: Directory for datasets.
  - `prices_all.csv`: The raw input data containing historical electricity prices.
  - `prices_wide_clean.parquet`: The standardized 'Silver' dataset generated after preprocessing.
- **`results/`**: Output directory. Contains:
  - `figures/`: Visualizations generated by the notebooks.
  - `models/`: Saved model weights and checkpoints.

### Notebooks
Sequential Jupyter notebooks used for data exploration, preprocessing, and iterative experimentation:
- **`00_data_preprocess.ipynb`**: Handles raw data loading, cleaning, capping extreme price spikes, and standardizing the data into a clean `.parquet` file.
- **`01_analysis_NSW.ipynb`**: Performs exploratory data analysis (EDA) and establishes baseline predictive models with metrics like sMAPE.
- **`02_feature_improvements_NSW.ipynb`**: Tests experimental and advanced feature engineering techniques (such as cyclical time encoding, wavelet decomposition, log transformations, and ensemble modeling) to systematically improve forecasting performance beyond the baseline R² scores.

### Core Modules (`src/`)
The underlying logic is abstracted into Python modules for reproducibility and cleaner notebooks:
- **`data_processing.py`**: Utilities for pivoting raw data, calculating statistical price caps (using the IQR method), and generating lagged/time-based features for the models.
- **`diffusion_model.py`**: A PyTorch-based conditional diffusion model designed for price forecasting. It includes an MLP Denoiser with Classifier-Free Guidance and methods for sampling with uncertainty estimation.
- **`trainer.py`**: The training loop for the diffusion model, managing the optimizer, loss computation, validation, and gradient clipping.
- **`utils.py`**: General utility functions, such as environment seeding for deterministic and reproducible model training.

## Getting Started

1. Ensure you have the necessary dependencies installed (e.g., `torch`, `pandas`, `numpy`, `jupyter`).
2. Place the raw `prices_all.csv` dataset in the `data/` directory.
3. Run `notebooks/00_data_preprocess.ipynb` to clean the data and generate the `prices_wide_clean.parquet` file.
4. Proceed to the analysis and feature improvement notebooks to explore the data and train the models.
